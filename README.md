# Local Bus Monitoring System

This is a PHP and MySQL based web application developed as part of a university lab project.  
The system allows users to view bus routes, manage favourite routes, and demonstrates database-driven web functionality.

---

## Features

- User Registration
- User Login (Session-based authentication)
- View Available Bus Routes
- Add Routes to Favourite
- View Favourite Routes
- User Logout
- Secure password hashing

---

## Technologies Used

- PHP
- MySQL (MariaDB)
- HTML
- XAMPP (Apache, MySQL)
- GitHub

---

## Database Design

The database is designed following an ER diagram and normalization principles.

Main tables include:
- user
- admin
- driver
- route
- stop
- route_stops
- schedule
- bus
- bus_location
- user_favourite_route

Primary keys and foreign keys are properly implemented.
